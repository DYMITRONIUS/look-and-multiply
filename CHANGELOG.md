# [0.9.0] - First Public Release

## ✨ Added

- Entity duplication by looking at them
- Personal cooldown timer
- Multiplayer synchronization
- Server-side player settings
- In-game settings GUI
- Per-player blacklist
- Configurable HUD
- Customizable messages
- Chat commands
- Localization support
- Support for dropped items
- Support for full item stacks
- Support for projectiles
- Support for modded features

## 🔧 Changed

- Improved timer synchronization
- Better multiplayer compatibility
- Cleaner GUI layout
- Improved settings system

## 🐞 Fixed

- Various timer bugs
- Multiplayer synchronization issues
- GUI refresh problems
- Settings saving issues
